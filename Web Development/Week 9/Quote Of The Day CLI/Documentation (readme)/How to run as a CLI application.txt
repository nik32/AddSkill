- Just open cmd from this folder.

- type the command (it will install 
							npm install -g .

  This installs your script “globally.” Any commands listed in the bin section of the package.json file will be made   available as command line applications. You can now run your script by typing "qod" at the command line!

- To uninstall your script, run the following command.

							npm uninstall -g quote_of_the_day
